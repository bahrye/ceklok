Petunjuk Penggunaan Aplikasi Ceklok

Panduan lengkap untuk setiap peran pengguna agar dapat memaksimalkan penggunaan aplikasi.

Untuk Semua Pengguna

Login Aplikasi

Gunakan Username atau Email Utama beserta Password yang telah terdaftar untuk masuk ke sistem.

Verifikasi Email Utama

Email Utama sangat penting untuk notifikasi dan fitur lupa password. Pastikan Anda melakukan verifikasi saat pertama kali diminta.

Lupa & Ubah Password

Gunakan tautan "Lupa Password?" di halaman login untuk reset via OTP. Anda juga dapat mengubah password kapan saja melalui menu profil.

Ajukan Perubahan Username

Jika perlu, Anda dapat mengajukan perubahan username melalui menu profil. Pengajuan akan ditinjau oleh Admin.

Untuk Guru

Mengisi Ceklok Harian

Pada halaman "Mulai Isi Kehadiran", pilih periode yang diinginkan, lalu isi Jam Masuk dan Jam Pulang. Jika tidak masuk, pilih alasan dari dropdown yang tersedia.

Melihat Riwayat

Gunakan tombol "Rekapitulasi Bulanan" atau "Kalender Kehadiran Tahunan" di dasbor untuk melihat riwayat kehadiran Anda.

Export Excel ke EMIS

Fitur ini tersedia di halaman "Mulai Isi Kehadiran". Ikuti petunjuknya untuk mengunduh file Excel yang dapat langsung diunggah ke aplikasi EMIS.

Untuk Admin

Dasbor Admin

Dasbor Admin adalah pusat kendali Anda. Dari sini, Anda dapat mengelola pengguna, daftar guru, kalender, dan melihat riwayat login.

Kelola Pengguna & Guru

Tambah, edit, atau hapus akun login pengguna dan daftar nama guru melalui menu "Kelola Akun Login Pengguna" dan "Kelola Daftar Nama Guru".

Kelola Kalender

Tetapkan hari libur nasional atau acara sekolah melalui menu "Kelola Kalender Pendidikan". Tanggal yang ditambahkan di sini akan otomatis menjadi hari libur di tabel ceklok.

Kunci Bulan Ceklok

Anda dapat mengunci data ceklok pada bulan tertentu untuk mencegah perubahan lebih lanjut. Fitur ini tersedia di Dasbor Admin.

Log Aktivitas (Superadmin)

Superadmin dapat memantau semua aktivitas penting yang terjadi di dalam sistem melalui menu "Lihat Log Aktivitas Pengguna".
