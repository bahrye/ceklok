Petunjuk Penggunaan Aplikasi Ceklok Guru

Selamat datang di Aplikasi Ceklok Guru. Berikut adalah panduan singkat penggunaan aplikasi ini.

Untuk Semua Pengguna

1. Login

  • Akses halaman login aplikasi.
  • Masukkan Username atau Email Utama Anda yang terdaftar.
  • Masukkan Password Anda.
  • Klik tombol "Login".
  • Jika Anda lupa password, klik tautan "Lupa Password?" dan ikuti instruksi untuk reset melalui OTP yang dikirim ke email utama Anda.

2. Email Utama

  • Saat pertama kali login (atau jika belum diatur), Anda akan diminta untuk mengatur Email Utama. Email ini sangat penting untuk fitur lupa password dan notifikasi.
  • Masukkan email aktif Anda dan lakukan verifikasi melalui OTP yang dikirimkan.
  • Anda dapat mengubah Email Utama melalui menu profil di pojok kanan atas.

3. Ubah Password

  • Anda dapat mengubah password melalui menu profil di pojok kanan atas.
  • Anda akan diminta memasukkan password lama dan password baru beserta konfirmasinya.

4. Logout

  • Untuk keluar dari aplikasi, klik ikon profil di pojok kanan atas, lalu pilih "Logout".
  • Sesi Anda juga akan otomatis berakhir setelah periode tidak aktif tertentu.

Untuk Guru

1. Mengisi Ceklok Harian

  • Setelah login, Anda akan diarahkan ke halaman Input Ceklok.
  • Nama Anda akan otomatis terpilih.
  • Pilih Bulan dan Tahun ceklok yang ingin diisi atau dilihat. Data untuk periode yang dipilih akan dimuat. Periode yang bisa dipilih dibatasi oleh pengaturan Admin dan bulan berjalan.
  • Untuk setiap tanggal hari kerja:
    • Isi Jam Masuk dengan format HH.MM (contoh: 07.15).
    • Isi Jam Pulang dengan format HH.MM (contoh: 14.30).
    • Jika Anda tidak masuk karena alasan tertentu (sakit, izin, dinas luar), pilih alasan dari dropdown Keterangan Libur/Alasan Tidak Masuk. Jika Anda memilih alasan, input jam masuk dan pulang akan otomatis dikosongkan/dinonaktifkan untuk hari itu.
    • Hari Minggu dan hari libur nasional/sekolah (yang sudah diinput Admin) akan ditandai dan input jam dinonaktifkan.
  • Setelah selesai mengisi, klik tombol "Simpan Data Ceklok".

2. Melihat Hasil Ceklok (Rekapitulasi)

  • Dari halaman Input Ceklok, Anda dapat mengklik tombol "Lihat Hasil Ceklok".
  • Ini akan membawa Anda ke halaman rekapitulasi absensi bulanan.
  • Halaman rekapitulasi akan dimuat, silahkan menunggu sampai benar-benar tampil.
  • Anda juga dapat mencetak rekapitulasi tersebut ke format PDF dengan mengklik "Cetak Rekap PDF".

3. Export Excel untuk EMIS

  • Di bagian bawah halaman Input Ceklok, terdapat fitur "Export File Excel".
  • Fitur ini berguna untuk mengunggah data absensi ke aplikasi EMIS.
  • Ikuti langkah-langkah yang tertera:

      Login terlebih dahulu ke EMIS

      Klik Absensi

      Cari bulan yang akan diisi, lalu klik Aksi >> Lihat Detail

      Silahkan klik Download Template

      Buka Template yang sudah di download lalu salin/catat Lock ID yang pertama tepat disamping tanggal satu (01) bulan template

      Geser hingga paling bawah halaman ceklok. Masukkan LOCK ID yang sudah di copy/salin/catat di kotak input LOCK ID

      Klik Simpan LOCK ID

      Selanjutnya, jika berhasil menyimpan, maka tombol Download File Excel bisa diklik. Silahkan diklik dan otomatis file Excel akan terdownload

      Di bawah download template sebelumnya di EMIS, silahkan klik Upload Excel

      Cari dan Pilih file yang sudah didownload di Ceklok manual

      Proses Upload Absensi berhasil dan semua ceklok absensi manual sudah masuk ke EMIS

  • PENTING: Pastikan data ceklok untuk bulan yang bersangkutan sudah lengkap dan benar sebelum melakukan export. Pastikan juga ketika mengupload file excel ke EMIS harus sama dengan bulan yang diisi di Ceklok Manual. Jika salah bulan, atau salah angka LOCK ID biasanya akan terjadi KESALAHAN/GAGAL. Jika seperti ini, maka silahkan ulangi kembali langkahnya dari awal.

4. Notifikasi Pengingat

  • Sistem akan mengirimkan email pengingat jika:
    • Ada absensi yang tidak lengkap (misalnya, hanya jam masuk yang terisi tanpa jam pulang, atau sebaliknya) pada hari kerja.
    • Ada hari kerja yang terlewat dan belum diisi absensinya sama sekali (baik jam maupun keterangan) setelah beberapa hari.
  • Mohon untuk menambah email utama dan memeriksa email Anda secara berkala dan segera lengkapi absensi Anda jika menerima notifikasi.
